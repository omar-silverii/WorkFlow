## Catálogo funcional de nodos validado

Recordá qué hace cada nodo:

```text
Cargar documento:
carga archivo/documento y expone input.*, texto y biz.* según DocTipo.

Validar condición:
evalúa campos reales y bifurca SI/NO.

Mandar tarea humana:
crea tarea a usuario/rol y permite ramas APTO/NO APTO con wf.tarea.resultado.

Solicitud HTTP:
ejecuta request y expone payload.status/body/json.

Consulta SQL:
ejecuta data.sql; SELECT expone sql.rows, rowCount, first, scalar, truncated.

Notificación interna:
crea avisos WF_Notificacion con lectura por usuario.

Guardar variable / Quitar variable:
state.vars; soporta simple, JSON, objetos, varios datos y remove anidado.

Esperar / Demora:
control.delay; pausa milisegundos reales.

Registrar log:
util.logger; guarda logger.last.*.

Manejador de Error:
util.error; marca wf.error=true y deja contexto/notificación.

Reintentar:
control.retry; opera sobre el nodo siguiente.

Archivo Leer / Escribir:
file.read/file.write; expone file.read.last* y file.write.last*.

Ejecutar otro workflow / Subflujo:
util.subflow; crea instancia hija y devuelve subflow.* y subflows.<alias>.*.
```
